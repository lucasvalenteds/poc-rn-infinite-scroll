# POC: Infinite Scroll

It demonstrate how to fetch new elements to a list when the scroll reaches the end of the screen.

## How to run

### Client

| Description | Command |
| :--- | :--- |
| Install dependencies | `npm install` |
| Run tests | `npm test` |
| Start Metro | `npm start` |
| Run app | `npm run android` |

### Server

| Description | Command |
| :--- | :--- |
| Install dependencies | `npm install` |
| Run tests | `npm test` |
| Start server | `npm start` |
